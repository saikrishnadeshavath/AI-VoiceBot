📘 AI Voice Bot (Script to Audio Generator)
🚀 Overview

This project is an AI-powered voice generation system that converts text scripts into high-quality audio using ElevenLabs Text-to-Speech.

It supports:

🎙 Multi-character conversations

🔊 Different voices per speaker

📄 Script-based input (.txt)

🎧 Automatic audio generation (MP3)

🧠 Features

Convert text scripts into realistic human-like speech

Detect and process multiple speakers (e.g., ARIA, Captain Webb)

Assign different voices dynamically

Generate full conversation audio

Works without OpenAI (only ElevenLabs required)

Compatible with Python 3.14 (no pydub required)
